you will need to create a file called ```.env``` in this folder.
This folder is just where i am keeping lal of the haar cascades.xml



It will need to look like the following:
This is the format:

```
#.env

eye_cascade = Path\To\This\Folder\haarcascade_eye.xml
eye_tree_eyeglasses_cascade = Path\To\This\Folder\haarcascade_eye_tree_eyeglasses.xml
frontalcatface_cascade = Path\To\This\Folder\haarcascade_frontalcatface.xml
frontalface_extended_cascade = Path\To\This\Folder\haarcascade_frontalcatface_extended.xml
frontalface_alt_cascade = Path\To\This\Folder\haarcascade_frontalface_alt.xml
frontalface_alt_tree_cascade = Path\To\This\Folder\haarcascade_frontalface_alt_tree.xml
frontalface_alt2_cascade = Path\To\This\Folder\haarcascade_frontalface_alt2.xml
frontalface_default_cascade = Path\To\This\Folder\haarcascade_frontalface_default.xml
fullbody_cascade = Path\To\This\Folder\haarcascade_fullbody.xml
lefteye_2splits_cascade = Path\To\This\Folder\haarcascade_lefteye_2splits.xml
licence_plate_rus_16stages_cascade = Path\To\This\Folder\haarcascade_licence_plate_rus_16stages.xml
lowerbody_cascade = Path\To\This\Folder\haarcascade_lowerbody.xml
```
